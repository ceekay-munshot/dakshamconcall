# Brand assets

## `munshot-logo.png`

Drop the Munshot logo here as **`munshot-logo.png`** (transparent PNG, roughly
square, ideally ≥ 256 px on the short side). It is used for the branding on the
**PDF report cover + page headers** and the **Excel export**.

If the file is absent, both exports fall back gracefully to a tasteful
**"MUNSHOT"** wordmark — nothing breaks. Add the PNG and the real logo appears
automatically on the next export (no code change needed).
